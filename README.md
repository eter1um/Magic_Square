# Магический квадрат

Компьютерная игра «Магический квадрат», разработанная на Python с использованием PyQt6 и JSON.

## Описание

«Магический квадрат» — логическая игра, в которой пользователь заполняет пустые клетки числами так, чтобы суммы в каждой строке, каждом столбце и на обеих диагоналях совпадали.

Проект разработан в рамках учебной практики.

## Скачать игру

Готовую Windows-версию можно скачать во вкладке **Releases** на GitHub.

1. Откройте раздел **Releases**.
2. Скачайте файл `MagicSquare.exe`.
3. Запустите файл двойным нажатием.

Дополнительная установка Python не требуется.

- отображение очков пользователя;
- окно с правилами игры;

- ввод и изменение чисел в пустых клетках;
- проверка правильности заполнения магического квадрата;
- вывод результата проверки;
- начисление очков за правильное решение;
- бонусные очки за быстрое прохождение;
- таймер во время игры;
- подсказки за очки;
- переход к новой игре;
- возврат в предыдущее окно;
- сохранение прогресса в JSON;
- окно настроек;
- смена темы интерфейса;
- смена языка интерфейса;
- регулировка громкости музыки и звуков;
- включение показа сумм строк и диагоналей
- фоновая музыка;
- звуковые эффекты;
- статистика пользователя;
- завершение работы программы.

## Управление

1. Нажмите **Новая игра**.
2. Выберите размер поля и сложность.
3. Заполните пустые клетки числами.
4. Нажмите **Проверить**.
5. При правильном решении игрок получает очки.
6. Очки можно потратить на подсказки и открытие новых размеров поля.

## Правила игры

Необходимо заполнить квадрат числами так, чтобы:

- суммы в каждой строке были одинаковыми;
- суммы в каждом столбце были одинаковыми;
- суммы на обоих диагоналях совпадали;
- каждое число использовалось только один раз.

## Сохранение данных

Прогресс пользователя сохраняется в файле `save.json`.

Сохраняются:

- количество очков;
- открытые размеры поля;
- статистика игр;
- выбранная тема;
- выбранный язык;
- громкость музыки и звуков.
