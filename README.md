# Магический квадрат

Компьютерная игра «Магический квадрат», разработанная на Python с использованием PyQt6 и JSON.

## О проекте

Игра представляет собой логическую головоломку, в которой нужно заполнить квадрат числами так, чтобы суммы в каждой строке, каждом столбце и на диагоналях были одинаковыми.

Проект разрабатывается в рамках учебной практики.

## Возможности

- главное меню
- страница с правилами
- выбор размера поля
- выбор уровня сложности
- динамическая генерация игрового поля
- поддержка полей 3x3 и 4x4
- система очков
- покупка новых размеров поля
- сохранение прогресса в JSON

## Стек технологий

- Python
- PyQt6
- JSON

## Структура проекта

- `game_gui.py` - интерфейс и основной игровой процесс
- `game_logic.py` - игровая логика и проверка решений
- `ui_config.py` - стили и параметры интерфейса
- `save.json` - сохранённый прогресс
- `levels/` - игровые уровни
